a tinder-inspired personal portfolio project i made for fun

## link
https://albertspersonalportfolio.vercel.app

## screenshots

## tech stack
- react
- vite
- tailwind css
- framer motion
- lucide react icons
- canvas-confetti
- embla carousel

## features
- embedded phone frame experience for the main card screen
- profile card with swipable photo carousel
- full profile page w/ photo carousel, about section, interests, projects, and contact links
- responsive design for mobile and desktop
- light/dark mode toggle

## how to run locally
1. install dependencies:
    - cd frontned
    - npm install

2. start the dev. server
    - npm run dev

3. open the local url shown in the terminal (will probably be http://localhost:5173)
